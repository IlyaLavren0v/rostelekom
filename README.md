# rostelekom
Это набор автотестов для следующих сценариев:
- Сценарий стандартной авторизации по логину и паролю.

1	Проверка наличия формы авторизации
2	Проверка наличия меню выбора типа аутентификации
3	Проверка наличия формы ввода номера/логина/почты/лицевого счета
4	Проверка наличия формы ввода пароля
5	Проверка наличия продуктового слогана
6	Проверка наличия вспомогательной информации для клиента
7	Проверка автоматического изменения табов при вводе данных

- Сценарий авторизации клиента по номеру телефона, кнопка "Номер"

1	Позитивная авторизация по номеру телефона с корректными данными
2	Негативная авторизация по номеру телефона с некорректным паролем
3	Негативная авторизация по номеру телефона с некорректным номером
4	Проверка перекрашивания элемента "Забыл пароль" при ошибке

- Сценарий авторизации клиента по почте, кнопка "Почта"

1	Позитивная авторизация по почте с корректными данными
2	Негативная авторизация по почте с некорректным паролем
3	Негативная авторизация по почте с некорректным адресом
4	Проверка перекрашивания элемента "Забыл пароль" при ошибке

- Сценарий авторизации клиента по логину, кнопка "Логин"

1	Позитивная авторизация по логину с корректными данными
2	Негативная авторизация по логину с некорректным паролем
3	Негативная авторизация по логину с некорректным логином

- Сценарий авторизации клиента по лицевому счету, кнопка "Лицевой счет"

1	Позитивная авторизация по лицевому счету с корректными данными
2	Негативная авторизация по лицевому счету с некорректным паролем
3	Негативная авторизация по лицевому счету с некорректным счетом
4	Проверка ограничения на ввод 12 символов в поле ввода лицевого счета
5	Проверка ограничения на ввод только чисел в поле ввода лицевого счета

- Сценарий Авторизация по временному коду:

1	Проверка отображения формы "Авторизация по коду" со всеми необходимыми элементами
2	Ввод корректного номера телефона/почты и нажатие кнопки "Получить код"
3	Ввод некорректного номера телефона/почты и нажатие кнопки "Получить код"
4	Проверка наличия шести отдельных полей для ввода кода подтверждения
5	Проверка отображения формы ввода кода с соответствующими элементами

Для подготовки виртуального окружения и запуска тестов необходимо выполнить следующие команды:

- python3 -m venv venv
- pip install requarements.txt
- pytest -v tests/


Часть тестов придется запускать отдельно, а часть не имеет возможности произвести проверку из-за отсутствия данных для авторизации.
На ресурсе появляется captcha, которая не позволяет в автоматическом режиме запускать все тесты.
